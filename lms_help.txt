Usage: lms server start [options]

Starts the local server

Options:
   -p, --port <port>            Port to run the server on. If not provided, the server
                                will run on the same port as the last time it was started.
   --bind <address>             Network address to bind the server to. Use "0.0.0.0" to
                                accept connections from the local network, or "127.0.0.1"
                                (default) for localhost only. Can also be set via the
                                LMS_SERVER_HOST environment variable.
   --cors                       Enable CORS on the server. Allows any website you visit to
                                access the server. This is required if you are developing
                                a web application.
